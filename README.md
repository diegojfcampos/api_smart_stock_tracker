## Run Locally

Clone the project

```bash
  git clone https://github.com/diegojfcampos/api_smart_stock_tracker.git
```

Go to the project directory

```bash
  cd api_smart_stock_tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
